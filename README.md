# UPI & Digital Payment Behavior Intelligence System

This project builds a **Dataset Personality Profiler** for messy fintech-style transaction data.

The system analyzes a digital payment dataset and automatically determines:

* Dataset complexity (linear vs nonlinear)
* Best performing ML model
* Most important financial features
* Noise level and outliers
* Behavioral clusters in user spending

## Dataset

A **very messy semi-synthetic UPI dataset** (~4000 transactions) with:

* missing values
* outliers
* skewed distributions
* noisy behavior
* overlapping user patterns

## Pipeline

1. Data Explorer
2. Model Training & Benchmarking
3. Dataset Complexity Detector
4. Feature Importance Analyzer
5. Dataset Clustering
6. Dataset Personality Report
